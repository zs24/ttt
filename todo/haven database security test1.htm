            3.1. In newhaven website, login as ‘zs24’ who doesn’t have ‘realname’ power. Try to run a test query, e.g. petInj in pet category, add pat.firstname in the field. The website denies this and returns this error.

Error executing query. (ERROR # 1143: SELECT command denied to user 'user1'@'ariespet.med.yale.edu' for column 'firstname' in table 'Patient' .)
